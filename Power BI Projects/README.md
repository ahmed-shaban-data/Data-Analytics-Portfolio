# 📊 Superstore Sales Dashboard | Power BI

## Project Overview
Interactive Power BI dashboard analyzing Superstore sales performance, profitability, customer segments, shipping modes, and regional trends.

## KPIs
- Total Sales: $2.3M
- Total Profit: $286K
- Total Orders: 5K
- Total Quantity: 38K
- Profit Margin: 12.47%

## Key Insights
- Technology generated the highest sales.
- West region achieved the highest profit.
- Consumer segment contributed the most revenue.
- Sales increased significantly from 2014 to 2017.

## Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling
