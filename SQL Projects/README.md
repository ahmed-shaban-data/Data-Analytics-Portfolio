

# 🛒 Walmart Sales Analysis | SQL

## Project Overview

This project analyzes Walmart sales data using SQL to identify revenue trends, branch performance, product performance, and customer behavior.

## SQL Skills Used

- SELECT
- WHERE
- GROUP BY
- ORDER BY
- Aggregate Functions (SUM, AVG, COUNT)
- SQL Filtering

## Business Questions Answered

1. What is the total sales revenue?
2. Which branch generates the highest revenue?
3. Which product line performs best?
4. What is the average customer rating?
5. Which products contribute most to overall sales?

## Key Insights

- Calculated total sales revenue.
- Identified top-performing branches.
- Analyzed revenue by product line.
- Measured average customer ratings.
- Generated business insights using SQL queries.
