# HR Analytics Dashboard

## Project Overview
Interactive HR Analytics Dashboard built using Power BI to analyze employee attrition, job satisfaction, salary distribution, and workforce demographics.

## KPIs
- Total Employees
- Attrition Count
- Attrition Rate
- Average Age
- Average Salary
- Average Job Satisfaction

## Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling

## Key Insights
- Research & Development department has the highest attrition count.
- Employees working overtime show higher attrition rates.
- Salary and job role impact employee retention.
- Workforce demographics reveal differences across departments and genders.

## Dashboard Features
- Interactive Filters
- KPI Cards
- Bar Charts
- Donut Charts
- Employee Demographics Analysis
